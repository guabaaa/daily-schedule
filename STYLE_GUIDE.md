## 배경색 개선

### 변경 사항 (글씨 가독성 향상)

#### 일정 블록
```css
/* Before */
opacity: 0.6 (계획) / 0.9 (실행)

/* After */
opacity: 1.0 (제거)
+ rgba(255, 255, 255, 0.4) 오버레이 추가
```

**효과**:
- 배경색이 40% 더 연해짐
- 글씨가 더 잘 보임
- 색상은 유지하면서 가독성 향상

#### 완료 배지
```css
/* 크기 증가 */
width: 20px → 26px
height: 20px → 26px
font-size: 12px → 15px

/* 위치 조정 */
top: 4px → 6px
right: 4px → 6px
```

## CSS 구조

### 일정 블록 레이어링

```css
.block {
  /* 베이스 레이어 - 카테고리 색상 */
  background-color: #FFE066;
}

.block::before {
  /* 오버레이 레이어 - 40% 흰색 */
  background-color: rgba(255, 255, 255, 0.4);
}

.content {
  /* 텍스트 레이어 - 검은색 */
  color: #212529;
}
```

**결과**: 밝고 부드러운 배경 + 선명한 텍스트
